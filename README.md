## Описание

Реализация HTML+CSS+JavaScript проект функционального календаря с навигацией по месяцам, выбором года и обработкой квери-параметра для установки начальной даты.
Задача взята из учебного практического курса.

## Требования к задаче
Инициализация календаря:
обработка квери-параметра: чтение параметра now в формате DD.MM.YYYY (например, ?now=12.04.2024)
Валидация даты: проверка корректности переданной даты
Fallback к текущей дате: использование текущей даты если параметр отсутствует или некорректен
Начальное отображение: календарь открывается на указанной или текущей дате
Инициализация выпадающего списка с годами: должны быть указаны опции ±20 лет от текущего года

Навигация по месяцам:
Кнопка “Previous”: переход к предыдущему месяцу
Кнопка “Next”: переход к следующему месяцу
Обновление отображения: автоматическое обновление сетки с днями месяца
Циклическая навигация: переход между годами при достижении января/декабря

Сетка календаря:
Сетка должно всегда состоять из 6 строк
Первая строка должна содержать хотя бы 1 день из отображаемого в даннымй момент времени месяца
Текущая дата в сетке месяца должна обязательно включать css-класс today
